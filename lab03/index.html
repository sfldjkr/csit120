Adding Style with CSS (Lab 03)

Published on September 18, 2025 by Safal Shrestha

I learned how to separate content from design using external CSS. Styling with fonts, colors, and spacing made my webpages look more professional and visually appealing.

Key Takeaways

  • Using external CSS for cleaner and reusable styling
  • Improving readability with spacing, fonts, and colors
  • Enhancing user experience with consistent design

See more CSS resources here: CSS Guide (MDN)

Reflection

Lab 02 Reflection: I learned how crucial it is to organize a webpage with purpose and clarity. HTML acts as the framework that gives structure to a page, ensuring information is presented in a logical and accessible way.

Lab 03 Reflection: I discovered how CSS adds the personality and design that make content engaging. Minor details, such as consistent spacing, font choices, and color schemes, significantly improve user experience. This gave me more confidence to design webpages that look both professional and functional.
