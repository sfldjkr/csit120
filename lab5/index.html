Adding Style with CSS (Lab 03)

Published on September 18, 2025 by Safal Shrestha

I learned how to separate content from design using external CSS. Styling with fonts, colors, and spacing made my webpages look more professional and visually appealing.

Key Takeaways

  • Using external CSS for cleaner and reusable styling
  • Improving readability with spacing, fonts, and colors
  • Enhancing user experience with consistent design

Responsive Design and Layout (Lab 05)

Published on October 19, 2025 by Safal Shrestha

In this lab, I learned how to make web pages responsive using CSS Flexbox and media queries. This allows content to adjust smoothly for both mobile and desktop users, creating a better viewing experience.

Key Takeaways

  • Creating flexible layouts using display: flex;
  • Using @media queries for responsive design
  • Testing designs across different screen sizes

Explore more about responsive layouts here: Responsive Design (MDN)
