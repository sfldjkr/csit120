MSSU Website Usability Evaluation

This evaluation analyzes the usability, accessibility, and design of the Missouri Southern State University (MSSU) website using web design and UI/UX concepts.

Website URL

The official MSSU website can be accessed at https://www.mssu.edu/.

Website Name

Missouri Southern State University (MSSU)

Target Audience

The MSSU website targets prospective students, current students, faculty and staff, parents/families, and alumni. Quicklinks address these groups with access to Lionet, Calendar, Course Catalog, Email, Blackboard, News, Giving, and Library.

This ensures that each type of user can access information relevant to their needs efficiently.

Author: Safal Shrestha | Published: 2025-11-02

Site Organization

The site uses a top navigation bar with main sections including About Us, Academics, Admissions, Financial Aid, Campus Life, and Resources. Each section contains subpages with detailed information. Academics lists all programs, including Associate’s, Bachelor’s, Graduate programs, and Certificates. The homepage highlights news, events, and important announcements.

This organization helps users locate information quickly, though some sections have many links that may slow first-time users or keyboard navigation users.

Author: Safal Shrestha | Published: 2025-11-02

CRAP Design Principle

The website demonstrates Contrast through white navigation links on a dark green background, making the menu items readable. Headings and subheadings use larger font sizes and bold formatting to clearly separate sections of content, helping users scan the site efficiently.

Author: Safal Shrestha | Published: 2025-11-02

Accessibility Audit Score

The MSSU website received an accessibility score of 65. Key issues include:

  • Landmarks should have a unique role or role/label/title combination (1 element)
  • Ensure all page content is contained by landmarks (1 element)
  • Ensure buttons have accessible names (1 element)
  • Ensure the page contains at least one level-one heading (1 element)

These issues may prevent screen reader users and keyboard users from navigating efficiently, highlighting areas for improvement.

Author: Safal Shrestha | Published: 2025-11-02

Effectiveness

The website supports users in completing tasks accurately. Prospective students can find admissions and program information, while current students can access resources like Lionet and Blackboard. Clear labeling reduces errors, but missing accessibility features may hinder screen reader users.

Author: Safal Shrestha | Published: 2025-11-02

Efficiency

Users can perform tasks quickly due to consistent navigation menus and logical content grouping. However, long dropdowns and dense link lists may slow first-time users or keyboard-only users slightly.

Author: Safal Shrestha | Published: 2025-11-02

Engagement

The website is visually engaging and professional. The green and white color scheme, clean layout, and campus images provide a positive experience. Interactive elements like calendars and social media links are user-friendly. Some low-contrast icons or missing alt text may reduce engagement for users with visual impairments.

Author: Safal Shrestha | Published: 2025-11-02

Recommendation for Improvement

To improve accessibility and user experience: landmarks should have unique roles or labels, buttons should include descriptive names, and all images should have meaningful alt text. Simplifying dropdown menus can enhance efficiency and usability for all users, including those relying on assistive technologies.
